# VocaBox Domain Context

## Vocabulary Import

Uploaded CSV data containing words, optional parts of speech, and one or more
meanings. A valid import supplies source material for a question set.

## Question Set

Prepared sequence of questions for one practice session. Preparation groups
repeated words, builds multiple-choice distractors, and assigns self-check or
scored question modes.

## Practice Session

One run through a question set. Owns current position, submitted attempts,
lives, streak, and navigation to results.

## Attempt

Recorded answer for one question. Multiple-choice attempts are scored;
self-check attempts record user input without changing score.

## Result Summary

Projection of completed attempts used by the result screen: score, percentage,
wrong attempts, and self-check count.
